Each file: `NNN-kebab-title.md`

```
# ADR-NNN: Title
**Date**: YYYY-MM-DD
**Status**: Active | Superseded by ADR-NNN | Deprecated

## Context
What situation led to this decision.

## Decision
What was decided.

## Consequences
What's better, what's worse, what to watch out for.
```
